# Assembly Simulator

For those who want to try assembly, our simulator tool
chain provides a compiler and a CPU simulator to run
basic assembly easily.

## How to Use This Tool

> [!note]
> This guide is written for python beginners, if you know
> how to build a venv, venv is recommended.

Firstly, install `numpy` by running following bash command:

```bash
pip install numpy
```

Check `.py` files in `./examples/` to explore some examples and
see practices in `./practices/` to try to solve problems.

See all commands in `./src/cpu.py`.
